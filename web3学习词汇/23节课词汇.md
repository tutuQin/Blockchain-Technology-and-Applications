### 📚 The DAO 事件与 Web3 核心词汇表

| 英文术语 (正面) | 中文释义 (背面) | 概念与记忆线索 |
| --- | --- | --- |
| **DAO** (Decentralized Autonomous Organization) | **去中心化自治组织** | 没有 CEO 和董事会，规则写在代码里，靠智能合约自动运转的组织形式。（The DAO 是史上首个著名案例） |
| **Smart Contract** | **智能合约** | 部署在区块链上的程序代码。满足条件自动执行，一旦上链不可篡改。 |
| **Code is Law** | **代码即法律** | 原教旨主义信仰：智能合约里写了什么就是什么，即使代码有 Bug 被利用也是“合规”的，人类不应干预。 |
| **Fallback Function** | **回退函数** | 智能合约里的特殊保底函数。当合约收到转账，或被调用的函数不存在时，会自动触发执行它。 |
| **Reentrancy Attack** | **重入攻击** | The DAO 被盗的罪魁祸首。利用“先转账、后扣余额”的漏洞，通过 Fallback 函数反复递归提款，掏空合约。 |
| **DoS** (Denial of Service) | **拒绝服务攻击** | 制造大量垃圾请求让系统瘫痪。以太坊曾试图用软分叉冻结黑客资金，但因矿工白费算力无 Gas 收入，引发了零成本的 DoS 风险。 |
| **Replay Attack** | **重放攻击** | 在 A 链上的合法交易广播，被原封不动拿到 B 链上广播，导致你在 B 链上的资产也被转走。 |
| **Soft Fork** | **软分叉** | 向下兼容的系统升级。旧节点依然能认可新节点产生的区块（通常是收紧规则）。 |
| **Hard Fork** | **硬分叉** | 向下不兼容的系统升级。旧节点认为新节点的区块非法，会导致社区永久分裂成两条链。 |
| **ETH** (Ethereum) | **以太坊** | The DAO 事件后，选择妥协、支持硬分叉回滚黑客资金并主导生态的链。 |
| **ETC** (Ethereum Classic) | **以太坊经典** | The DAO 事件后，坚持“代码即法律”、拒绝硬分叉修改账本的原链（黑客的钱在上面依然合法）。 |
| **Chain ID** (EIP-155) | **链标识符** | 给每条区块链分配的一个唯一编号。在交易签名时加入它，用来彻底防御“重放攻击”。 |
